Hugo

Hugo [w] [d] ist eine freie Software zur schnellen Erzeugung statischer Websites <static website generator>.

  • Hugo hat einen HTTP-Server [w] zur lokalen Ansicht der erzeugten HTML-Dateien eingebaut.
  • Hugo ist in der Programmiersprache Go [w] geschrieben.
  • Hugo erzeugt die Webseiten aus HTML- [w] oder Markdown-Dateien [w].

Hugo und git CLI-Befehle

Auf den Ordner mit dem Projekt gehen.

cd documents/programmierung/github/quickstart

Einen neuen Post anlegen.

hugo new posts/my-first-post.md

Neue Seite auf den lokalen Server bringen.

hugo server

hugo server -D

Seiten im public Ordner aktualisieren.

hugo

Neue Dateien zu git hinzufügen.

git add .

Die Änderungen lokal commiten.

git commit -m ‘Added files’

Die Änderungen auf das remote GitHub schieben.

it push -u origin main

Auf GitHub “Action” aktivieren und Seite veröffentlichen.
