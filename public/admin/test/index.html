Klassen

Eine Klasse <class> {d} ist ein Typ für referenzierbare Objekte zur Datenkapselung von Eigenschaften <properties> und Methoden <methodes>. Für die Deklarierung wird das Schlüsselwort class genutzt.

// Allgemein
class EineKlasse {
   // Definition des Inhalts der Klasse
}

// Beispiel
class Person {
    // Property
    var name: String = "volker"
    // Methode
    func gruesse () -> String {
        return "Hallo " + name
    }
}
