---------------------------------------------------
Auflistung der Hugo Variablen
Link: https://gohugo.io/functions/hugo/
---------------------------------------------------
BuildDate:
the compile date of the current Hugo binary formatted per RFC 3339 (e.g., 2023-05-23T08:14:20Z).
{{ hugo.BuildDate }}

CommitHash:
the Git commit hash of the Hugo binary (e.g., 0a95d67...).
{{ hugo.CommitHash }}

Deps:
See hugo.Deps.
{{ hugo.Deps }}

Environment:
the current running environment as defined through the --environment CLI flag (e.g., development, production).
{{ hugo.Environment }}

Generator:
renders an HTML meta element identifying the software that generated the site (e.g., ).
{{ hugo.Generator }}

GoVersion:
the Go version used to compile the Hugo binary (e.g., go1.20.4).
{{ hugo.GoVersion }}

IsExtended:
returns true if the Hugo binary is the extended version.
{{ hugo.IsExtended }}

IsProduction:
returns true if hugo.Environment is set to the production environment.
{{ hugo.IsProduction }}

Version:
the current version of the Hugo binary.
{{ hugo.Version }}

WorkingDir:
the project working directory (e.g., /home/user/projects/my-hugo-site).
{{ hugo.WorkingDir }}
